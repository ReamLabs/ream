# ream

A Rust implementation of the Ethereum Beam Chain specification.

```bash
$ ream --help
```
```txt
Usage: ream <COMMAND>

Commands:
  beacon_node     Start the node
  validator_node
  help            Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
